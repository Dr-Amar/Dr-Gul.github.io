About Me

I am a Data Scientist and Geoscientist with a Ph.D. specializing in Artificial Intelligence, Machine Learning, and Geospatial Analysis. I am passionate about solving geoscientific problems through innovative AI and big data solutions.

Projects

Mineral Alteration Mapping of Arabian Shield KSA

Using remote sensing and AI/ML algorithms to map mineral alterations over a large area of the Arabian Shield.

Machine Learning-Driven Classification of Pb-Zn Ore Deposits

Developed machine learning models to classify Pb-Zn ore deposits using pyrite trace elements and isotopic signatures.

Big Data Mining on Galena Geochemistry

Applied big data mining techniques to geochemical data from Galena to discriminate metallogenic patterns in ore deposits.

Exploring Auriferous VMS Prospective Zones in Egypt

Using integrated remote sensing data, field studies, EPMA chemistry, and XRD analysis to explore auriferous zones in Egypt.

Geospatial Mapping of Pb-Zn Ore Deposits in the Central Tethyan Domain

Conducted geospatial analysis to map Pb-Zn ore deposits in the Central Tethyan metallogenic domain.

Research

Machine Learning in Geochemistry

Research on applying machine learning algorithms in geochemical data analysis to predict ore genesis and mineral deposits.

Big Data in Mineral Exploration

Using big data techniques to mine geochemical data and improve mineral exploration processes.
